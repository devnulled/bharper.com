My Software Blog: devnulled

devnulled is a blog which caters to software development related issues with a pro-unix and open standards slant. I originally devnulled in early 2003 to write about various technical challenges I came across and how I fixed them. Over the years the focus and changed a bit here and there, but overall I still primarily write about various software related issues.

Over the years devnulled has been featured on popular sites of the times including Slashdot, Digg, O'Reilly Hacks (now MAKE Magazine), del.icio.us/popular, and the Indeed Blog. At its peak, it regularly had over 2,000 unique visitors per day. I haven't been maintaining it as well as I used to, but aim to start doing that again once I ditch Wordpress.

 

 

Books

After working on my first book, Inside ColdFusion MX, as a Technical Editor, I quickly learned that I would likely never be a book author as it takes A LOT of effort to write a book and keep-up with deadlines while also maintaining a full-time career. However, I did really enjoy being a Technical Editor and have been getting back into it again recently.

 

Making Java Groovy

Technical Reviewer

Making Java Groovy is an upcoming title from Manning Press about enhancing existing Java projects with Groovy. I served as a Technical Reviewer for many chapters in this book.

 

Inside ColdFusion MX

Technical Editor

Inside ColdFusion MX was a New Riders Press book about the first version of ColdFusion to be based on the J2EE platform, ColdFusion MX. This book was the first to market about ColdFusion MX and also maintained a 4.5 star rating on Amazon.
